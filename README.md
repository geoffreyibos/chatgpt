# Sports Crush 🏆

Un mini-jeu de match-3 inspiré de Candy Crush, mais avec des équipements sportifs au lieu de bonbons !

## 🎮 Comment jouer

1. Ouvrez `index.html` dans votre navigateur
2. Cliquez sur deux équipements sportifs adjacents pour les échanger
3. Alignez 3 équipements identiques ou plus horizontalement ou verticalement
4. Gagnez des points pour chaque alignement réalisé
5. Vous avez 30 mouvements pour obtenir le meilleur score possible !

## 🏀 Équipements sportifs

- ⚽ Football
- 🏀 Basketball  
- 🎾 Tennis
- 🏈 Football américain
- ⚾ Baseball
- 🏐 Volleyball

## 🛠️ Technologies utilisées

- **HTML5 Canvas** - Pour le rendu graphique du jeu
- **CSS3** - Pour le style et les animations
- **JavaScript ES6** - Pour la logique du jeu
- **Emojis** - Pour les équipements sportifs (compatible tous navigateurs)

## ✨ Fonctionnalités

- Grille de jeu 8x8
- Système de score avec bonus pour les alignements longs
- Animations fluides pour les mouvements et suppressions
- Effet de gravité quand les pièces tombent
- Détection automatique de fin de partie
- Design responsive (mobile-friendly)
- Interface en français

## 🚀 Installation

Aucune installation requise ! Simplement :

1. Clonez ou téléchargez ce repository
2. Ouvrez `index.html` dans votre navigateur web
3. Commencez à jouer !

## 🎯 Règles de scoring

- Alignement de 3 : 30 points
- Alignement de 4 : 45 points  
- Alignement de 5 : 60 points
- Alignement de 6+ : 75+ points
- Cascades (combos) donnent des points supplémentaires

Amusez-vous bien ! 🎉
